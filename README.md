# SunriseApp

日の出時刻検索アプリ(Udemyの学習課題)

## できること

- 地名で日の出時刻を検索できる
    - 日の出時刻の取得は OpenWeatherAPI を使用している

## Getting Started

- `.env.example` をコピーして `.env` を作成する。
    - 参考: https://qiita.com/nnsnodnb/items/9344a1955c5cf50e61af
- API_KEY は以下で取得する。
    - [OpenWeatherMap](https://home.openweathermap.org/api_keys)
